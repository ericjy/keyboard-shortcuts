# IntelliJ IDEA Shortcuts  

|            Action           |           Mac Keys       |         Linux Keys       |   
|-----------------------------|--------------------------|--------------------------|
| Search any action           | ⌘ + ⇧ + A                | Ctrl + Shift + A         |
| Go to class                 | ⌘ + N                    | Ctrl + N                 |
| Expand selection            | ⌘ + W                    | Ctrl + W                 |
